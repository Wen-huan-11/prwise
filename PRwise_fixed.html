PR #42 · user.ts:45-48
-function getUser(id) {
- const data = fetch(`/users/${id}`);
- return data.json();
-}
+function getUser(id: string) {
+ if (!id) throw new Error('id is required');
+ const data = await fetch(`/users/${id}`);
+ return data.json() as User;
+}
AI
审查结果
✕
缺少 await
fetch() 返回一个 Promise。没有 await，函数返回的是 Promise 而不是实际数据。
⚠
缺少空值检查
id 参数可能为 null 或 undefined。在 fetch URL 中使用前添加校验。
✓
建议：添加类型注解
function getUser(id: string): Promise<User> 可以提高类型安全性。
