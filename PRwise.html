AI
Review Results
✕
Missing await
fetch() returns a Promise. Without await, the function returns a Promise instead of the actual data.
⚠
Missing null check
id parameter could be null or undefined. Add validation before using it in the fetch URL.
✓
Suggestion: Add type annotation
function getUser(id: string): Promise<User> would improve type safety.
